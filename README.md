# 🚀 Portfolio Django

Portfolio personnel développé avec Django, présentant mes compétences, projets et parcours en Mécatronique, développement web et cybersécurité.

![Python](https://img.shields.io/badge/Python-3.x-blue)
![Django](https://img.shields.io/badge/Django-5.x-green)
![License](https://img.shields.io/badge/License-MIT-lightgrey)

## 📋 Aperçu

Portfolio dynamique avec interface d'administration Django permettant de gérer facilement :
- Compétences techniques (avec niveau et description détaillée)
- Projets (avec technologies, liens GitHub/démo, images)
- Expériences et formations

Chaque section dispose d'un système **« voir plus »** interactif (modales) pour afficher des détails sans surcharger la page principale.

## 🛠️ Technologies utilisées

- **Backend** : Django 5.x, Python 3
- **Frontend** : HTML5, CSS3, JavaScript (vanilla)
- **Base de données** : SQLite (dev) — migrable vers PostgreSQL
- **Admin** : Interface Django Admin native
